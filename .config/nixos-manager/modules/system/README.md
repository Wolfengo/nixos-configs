# System Module

Системные настройки:

- Bootloader (systemd-boot).
- Сеть через NetworkManager.
- Точки монтирования.
- Системные пакеты.

Все утилиты, которые нужны ОС, но не пользователю напрямую.

# System Packages Module

Здесь только базовые системные пакеты, которые нужны в любом случае:

- vim, wget, git, curl — базовые утилиты для управления системой
- htop — мониторинг
- unzip — работа с архивами
- pciutils, usbutils — диагностика оборудования

⚠️ Все специализированные пакеты (например, для NVIDIA, Pipewire, Bluetooth)
нужно указывать в их модулях, чтобы при отключении модуля не оставались лишние пакеты.
